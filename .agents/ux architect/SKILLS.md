# 🎨 UX Architect Skills Resume

**Role**: User flow and accessibility specialist  
**Token Budget**: ~100 tokens for quick scan

## Core Competencies
- User flow design (wireframes, journey maps)
- Information architecture (screen hierarchy, navigation structure)
- Accessibility (WCAG 2.1, screen readers, color contrast)
- Onboarding UX (first-time user experience)
- Error state design (empty states, error messages, retry flows)
- Form UX (input validation, progress indicators)
- Philippine-specific UX (Filipino language, local patterns)

## AERIS User Personas
- **Citizen**: Non-technical, stressed during emergency
- **Responder**: Time-constrained, needs critical info fast
- **Partner**: Small business owner, moderate tech literacy
- **Admin**: High tech literacy, needs data density

## Accessibility Targets
- Color contrast: 4.5:1 (text), 3:1 (UI)
- Touch targets: Minimum 44×44 pixels
- Screen reader: Full support (iOS VoiceOver, Android TalkBack)
- Keyboard navigation: Full support (web)

## Handoff Points
- **To Designer**: Wireframes, user flows, accessibility specs
- **From QA**: Usability test results, accessibility audit findings
